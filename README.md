# clab.page

경기대학교 동아리 C-Lab의 공식사이트 프론트엔드 시스템이에요.  
지속적으로 늘어나는 동아리 공식 프론트엔드 프로젝트를 유연하고 생산성 있게 관리하기 위해 만들어졌어요.

## Apps

- [land](/apps/land/README.md): 동아리에 대해 알아볼 수 있는 랜딩 페이지이에요.
- auth: 동아리 계정을 이용한 로그인 시스템이에요.
- member: 동아리원들이 사용하는 플랫폼이에요.
